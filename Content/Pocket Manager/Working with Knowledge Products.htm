Working with Knowledge Products

Knowledge Products (KPs) are documents housed with the CIC. Performing a search yields KPs that can be stored in a pocket to aid in research and to organize source material for new report creation. KPs can come in many formats, including PDF files, Microsoft Word files, Microsoft PowerPoint files, html files, and graphic .jpg files.

To add a KP to a pocket

  1. From the home screen, perform a search. Use the search icon or the Navy Medicine to go to the home screen.

  2. In the main pane, click a KP in the search results to open it in the Preview pane on the right.

  3. A blue bar will appear at the bottom of the main pane. Click the pocket icon , then select a pocket into which to add the KP, or create a new pocket.

To add a note to a KP

  1. From the home screen, select the Pocket Manager icon .

  2. In the main pane, click the row of the KP to which you want to add a note.

  3. A blue bar will appear at the bottom of the main pane. Click the bubble icon , then click the pocket row once more. A new note row will be added. Select the Note row then select edit on the blue bar at the bottom. Add your note. Hit enter. The note is added.

To highlight a KP excerpt and save to a pocket
  1. If you are using a KP directly from a search result, select the KP from search results to enter Preview mode on the right side of the screen.

  2. If you have previously saved the KP to a pocket, select the Pocket Manager icon , and click the row for the KP to enter Preview mode on the right side of the screen.

  3. In the Preview pane on the right side of the screen, highlight a passage/excerpt by holding down left click on the mouse and dragging the mouse pointer over the text you wish to highlight. Once highlighted, you can release left click.

  4. With the passage/excerpt highlighted, a pop-up bubble icon appears over the text. Click the pop-up bubble icon.

  1. A window will appear that will allow you to select a pocket into which to place your highlighted excerpt. Select the pocket you wish to add it to (or select the option to create a new pocket), add a note if you desire, and click save. You can also click Remove to undo your highlighted section.

To add a note to a KP excerpt

  1. From the home screen, select the Pocket Manager icon .

  2. In the main pane, click the row of the KP excerpt to which you want to add a note.

  3. A blue bar will appear at the bottom of the main pane. Click the bubble icon , then click the pocket row once more. A new note row will be added. Select the Note row then select edit on the blue bar at the bottom. Add your note. Hit enter. The note is added.
